# A股实时行情腾讯 Fallback

> 日期：2026-04-01
> 状态：✅ 已完成

## 背景
东方财富 API 从服务器无法访问（Connection refused / timeout）。腾讯股票接口 `qt.gtimg.cn` 直连可用，作为 fallback。

## 实现方案

### 修改文件
- `fetchers/daily_quote.py` — `fetch_a_spot()` 方法 + 新增 `_fetch_a_spot_tencent()`
- `tests/test_fetchers/test_daily_quote.py` — 新增 3 个测试用例

### 逻辑流程
1. 先尝试 `ak.stock_zh_a_spot_em()`（东方财富）
2. 任何异常 → fallback 到 `_fetch_a_spot_tencent()`
3. 腾讯接口使用 `ak.stock_info_a_code_name()` 获取 A 股代码列表（不依赖东方财富）
4. 批量查询 `https://qt.gtimg.cn/q=sh600519,sz000001,...`，每批 700 只（URL 长度 <8000 字节）

### 腾讯接口字段映射
腾讯返回 `~` 分隔字段，关键字段：
| 索引 | 含义 | 单位 |
|------|------|------|
| [1]  | 名称 | — |
| [2]  | 代码 | — |
| [3]  | 最新价 | 元 |
| [4]  | 昨收 | 元 |
| [5]  | 今开 | 元 |
| [6]  | 成交量 | 手 |
| [31] | 涨跌额 | 元 |
| [32] | 涨跌幅 | % |
| [33] | 最高 | 元 |
| [34] | 最低 | 元 |
| [38] | 换手率 | % |
| [44] | 流通市值 | 亿 |
| [45] | 总市值 | 亿 |
| [46] | 市净率 | — |
| [52] | 市盈率 | — |
| [57] | 成交额 | 万 |

### 单位转换（腾讯 → 东方财富格式）
- 成交量: 手 × 100 → 股
- 总市值/流通市值: 亿 × 1e8 → 元
- 成交额: 万 × 1e4 → 元

### 输出列
与东方财富 `ak.stock_zh_a_spot_em()` 一致：代码, 名称, 最新价, 涨跌额, 涨跌幅, 成交量, 成交额, 振幅, 最高, 最低, 今开, 昨收, 换手率, 市盈率-动态, 市净率, 总市值, 流通市值

## 验证结果
- pytest: 96/96 通过
- 实际拉取: 5193 只 A 股，含总市值/市净率/市盈率各 5193 条非空
- 振幅字段在腾讯 fallback 中为 None（腾讯不直接提供振幅）

## 注意事项
- 腾讯接口 **支持** 总市值和流通市值（验证过，亿为单位）
- `振幅` 字段腾讯不直接提供，fallback 时为 None（transform 层已兼容）
- 未修改港股/美股相关代码
