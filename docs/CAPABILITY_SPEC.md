# Stock Data System — 数据能力说明

## 系统概述

本系统覆盖 **A 股（5,493 只）、港股（2,743 只）、美股（518 只 S&P 500）** 三个市场，提供基本面数据、实时行情和财务指标查询。

---

## 一、数据范围

### 1.1 覆盖市场

| 市场 | 股票数 | 行业分类 | 分类标准 |
|------|--------|---------|---------|
| A 股 | 5,493 | ✅ 申万一级行业（31 类） | 申万 |
| 港股 | 2,743 | ✅ 港交所行业分类 | 港交所 |
| 美股 | 518 | SIC Code | SEC |

### 1.2 财务数据

| 数据 | A 股 | 港股 | 美股 |
|------|------|------|------|
| 利润表 | ✅ 3,620 只，1988 年至今 | ✅ 2,662 只，1996 年至今 | ✅ 503 只，2006 年至今 |
| 资产负债表 | ✅ | ✅ | ✅ |
| 现金流量表 | ✅ | ✅ | ✅ |
| TTM 指标 | ✅ | ✅ | ✅ |

**TTM（Trailing Twelve Months）指标已预计算，包含：**
- TTM 营收、TTM 净利润、TTM 归母净利润
- TTM 经营现金流（CFO）
- TTM 自由现金流（FCF = CFO - 资本支出）

### 1.3 行情数据

- **A 股/港股**：PE（TTM）、PB、总市值、流通市值、涨跌幅、成交额、换手率
- **美股**：PE、PB、总市值、涨跌幅、成交额（单位：美元）
- ⚠️ 目前仅最新一天的行情快照，暂无历史日线

## 二、能做什么筛选

### 2.1 价值筛选

| 指标 | 数据来源 | 说明 |
|------|---------|------|
| PE（TTM） | 行情 / TTM 视图 | 市盈率，衡量估值 |
| PB | 行情 | 市净率 |
| FCF Yield | 物化视图 `mv_fcf_yield` | 自由现金流收益率 = FCF / 市值 |
| 股息率 | ❌ 暂无 | 分红数据尚未同步 |

**示例查询：** FCF Yield > 10%、市值 > 500 亿、PE > 0、剔除金融/地产/ST

### 2.2 质量筛选

| 指标 | 计算方式 | 数据来源 |
|------|---------|---------|
| ROE | 归母净利润 / 归母净资产 | 利润表 + 资产负债表 |
| 毛利率 | 毛利润 / 营收 | 利润表 |
| 净利率 | 净利润 / 营收 | 利润表 |
| 营收增速 | 同比对比 | 多年利润表 |

**示例查询：** 连续 3 年 ROE > 15%（A 股 131 只、港股 94 只）

### 2.3 成长筛选

| 指标 | 计算方式 | 数据来源 |
|------|---------|---------|
| 营收 YoY | (本期营收 - 去年同期) / 去年同期 | 利润表 |
| 净利润 YoY | 同上 | 利润表 |
| CFO / 净利润 | 经营现金流 / 净利润 | 现金流量表 + 利润表 |
| FCF / 净利润 | 自由现金流 / 净利润 | TTM 视图 |

### 2.4 行业筛选

- 按 **申万一级行业** 筛选 A 股（31 个行业）
- 按 **港交所行业** 筛选港股（与 A 股分类不同）
- 支持行业内按指标排序
- 支持排除特定行业（银行、证券、地产等）

### 2.5 组合筛选（可组合使用）

```
市场 = A股
行业 ≠ 银行、证券、保险、地产、ST
市值 > 500 亿
PE（TTM）> 0 且 < 30
ROE 连续 3 年 > 15%
FCF Yield > 5%
```

## 四、数据局限

| 限制 | 说明 |
|------|------|
| ❌ 无历史日线 | 仅有最新一天行情快照，无法做技术分析或回测 |
| ❌ 无分红/股息数据 | 无法计算股息率、分红率 |
| ❌ 无股本变动 | 无法精确计算每股指标历史变化 |
| ⚠️ 2025 年报不完整 | A 股仅 268 只发布（截止 2026-04-07），预计 4 月底出完 |
| ⚠️ 港股行业回填中 | 78% 已完成，完成后可按行业筛选港股 |
| ⚠️ A 股/港股行业分类不统一 | 申万 vs 港交所，跨市场行业对比需映射 |
