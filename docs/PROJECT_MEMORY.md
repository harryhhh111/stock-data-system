## 概况

- **路径**: /root/projects/stock_data
- **技术栈**: Python 3.12 / PostgreSQL 16 / akshare / 东方财富 API / SEC EDGAR
- **架构**: fetchers/ → transformers/ → db.py → sync.py (CLI) → scheduler.py (定时)
- **数据规模**: 6282 只股票，~30 万行财务报表数据（A股+港股）
- **覆盖市场**: CN_A（A股）、CN_HK（港股）、US（美股，目前仅 S&P 500 共 503 只）
